Rows¶

class dbapix.row.Row(raw, cur)[source]¶

A row in a query result.

Rows are tuples, but they have been augmented to behave like dicts:

for row in cur.execute('''SELECT foo FROM bar'''):
    print(row[0])
    print(row['foo']) # The same thing.

Warning: checking if a string is in a row via 'foo' in row will check for BOTH the key 'foo' and the value 'foo'! If you need to be explicit, then you should look in the various views:

'foo' in row.viewkeys()
# of
'foo' in row.viewvalues()
